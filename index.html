Privacy Policy

Last updated: March 27, 2026

Overview

This application ("WHOOP Sleep Tracker") is a personal, single-user integration that connects WHOOP wearable data to a private Telegram bot. It is not a public service and is not available for general use.

Data Collection

This application accesses the following data from your WHOOP account via the official WHOOP API:

  • Sleep data — sleep duration, sleep stages, bed/wake times, sleep performance, consistency, efficiency, respiratory rate, disturbances, sleep debt
  • Recovery data — heart rate variability (HRV), resting heart rate, recovery score, SpO2, skin temperature

No other WHOOP data is accessed (no workout, strain, or profile data).

Data Usage

The collected data is used solely to:

  • Display sleep metrics within a private Telegram bot conversation
  • Provide context for mental health tracking and analysis

Data Storage

All data is stored locally on the application owner's private server in an SQLite database. Data is never transmitted to third parties, sold, or shared. OAuth tokens are stored locally and used exclusively for API authentication.

Data Retention

Data is retained indefinitely on the local server for personal tracking purposes. The user may delete all data at any time by removing the local database file.

Third Parties

This application does not share any data with third parties. The only external communications are:

  • WHOOP API — to retrieve sleep and recovery data
  • Telegram Bot API — to deliver messages to the bot owner

Security

Access to the Telegram bot is restricted to a single authorized user ID. OAuth credentials are stored securely on the private server.

Contact

This is a personal project. For any questions, contact the application owner directly.

Changes

This privacy policy may be updated at any time. Changes take effect immediately upon posting.
